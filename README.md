Installation:
1) Clone repository:
    mkdir <project name>
    cd <project name>
    git clone https://github.com/snazh/blog .

2) Set Up a Virtual Environment
    python -m venv venv
    .\venv\scripts\activate
3) Install dependencies:
    pip install -r requirements.txt
4) Launch Local Server:
    python manage.py runserver
